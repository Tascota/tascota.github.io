好评如潮组

負けヒロインが多すぎる！ / 败犬女主太多了！

Bangumi 8.03 / Tascota 8.0

本季度毫无疑问的口碑与人气双巅峰。

青春感与夏日感是本作的最突出特色，在这方面表现的完美可以从一处剧情反衬出——如果社友们还记得的话，在第二集开头，温水君和柠檬完美地复刻了历史上一次次的更衣室事变。从表面上看，这一情节不过是轻小说中惯有的卖肉桥段而已，但从某种程度上来说却与本作的主题基调相当格格不入。在其他作品里平平无奇的桥段，在本作中却显得违和，算是一种奇怪的对比。


【推しの子】 第2期 / 【我推的孩子】 第二季

Bangumi 7.38 / Tascota 7.0

《推子》第二季没有了第一季开头那样的大爆特爆，但动工依然献上了一份优秀的答卷。总体上来说，《推子》第二季依然是瑕不掩瑜的作品——与第一季后半部分类似，第二季依旧有时被人诟病又有时大放异彩。这种评论区的稳定最令人安心，甚至不需要做过多的吐槽。


出人意料组

真夜中ぱんチ / 深夜冲击

Bangumi 7.38 / Tascota 7.5

《深夜冲击》是 PA 社这一季度的三开之一，同时也是 PA 社最传统艺能的原创动画。

自从 PA 社在 10 年代前期以“工作少女”系列为首的打工原创动画，在 2015 年以广受好评的《白箱》结束之后，PA 社后来的原创动画一直未有特别出色的表现。例如，在与 Key 和麻枝准合作的作品里，《夏洛特》褒贬不一，《成神之日》更是给大家喂了坨大的。

不过就算是这样，PA 社也一直没放弃原创——“众所周知”，PA 的制作路线是这样的：接两三部动画的活赚钱，然后做一部原创给自己爽一爽。而在改编动画中，也不乏去年的《跃动青春》这样的佳作。

从 2022 年的两部神金动画（《派对浪客诸葛孔明》以及《秋叶原冥途战争》）之后，PA 社越发跳出自己青春、工作之类的原创动画标签，开始尝试新的风格。《深夜重拳》便是在这样的环境下创造出来的。

Mayopan 总体上看很像上上个季度的《水母》。同样是做 Youtuber，同样被踢出原队伍，同样有女女关系性，但本作对互联网生态的描写却比《水母》真实得多，与之相比，本作唯一不足的可能是人设风格不够讨喜而已，这也是本作几乎唯一的短板。

在当下的互联网时代，人人都是神金，人人都不是神金。神人频出的时代里，Mayopan 展现的真实情感（虽然这里只是女同情感）才最打动人心。

另外，本作的 OP、ED 都相当值得一听，羊宫妃那还唱了一首比在 MyGO 唱得好的 ED。


義妹生活 / 义妹生活

Bangumi 7.27 / Tascota 7.5

从名字上看比较像是滥俗轻小说，但实则并非如此。妹系动画既已从《妹非妹》之后就一蹶不振，而在当下，Kaju 和 Yuki 这种流行的妹系角色都只作为配菜出现了（此事在大春物中亦有记载）。

本作的侧重点从头到尾便与传统的妹系作品——尤其是伏见司的——不同，侧重于“义妹”这一家庭角色的另一面：义妹本质上是一个新的家庭成员。本作中男女主迎来的是单亲家庭的重组，由此带来的男女主间心之壁的逐渐消解、关系的逐渐拉近。其他作品并不是不会提到这一点，但本作不只是用这一点制造泪点、锦上添花，而是将其置为主题，试图深入探讨这一问题。本作中男女主各自的心理描写尤其丰富，通过日记等等方式，大量的角色心理独白贯穿全片，非常特别。

另外，本作尤其值得一看的是电影感。监督对于经费短缺的解决方案是活用演出手段，发文艺青年病。其实这种方案的好坏见仁见智，至于我本人则并不排斥这种特别的风格。

总之，《义妹生活》算是一部优点与缺点都很明显的动画，对得上电波的话值得一看。


先輩はおとこのこ / 前辈是男孩子

Bangumi 6.40 / Tascota 6.0

又是一部看似噱头的作品。

如果各位不知道的话，本作的开头是这样的故事：女主向女装的男主告白，男主却婉言拒绝，同时披露了自己女装的事实。结果女主更兴奋了。

开播前，我大胆预言 Makoto 一定会接替 Mahiro 的位置，没想到居然没成。

这部动画的特别之处在于，它是由大名鼎鼎的 Project No.9 制作的，但却意外地能看。柳监督通过把全片几近一半的 cut 换成 Q 版，成功保证了正常画风的人脸几乎没有崩。

本作表面上是 BG、BL 和 GL 大乱炖，但实际上是掺杂了性别问题的三角恋故事，其刻画可称及格水平，但对于 P9 来说已经相当不错了。


令人感叹组

ATRI -My Dear Moments- / 亚托莉 -我挚爱的时光-

Bangumi 6.28 / Tascota 6.5

Atri 和《千恋万花》以及《饿殍》一道，被称为当代 galgame 界三大原神。现在看，《原神》本神动画两年前出了个 PV 就没影了，而我们的 galgame 界原神现在动画化，名气和质量总得占一个吧？

Atri 的剧本依然是著名的花田大老师写的。其实我本人对花田并没有太多的怨言，但无奈各位在吹三对花田老师印象深刻，以至于在 Atri 这，成功让芙莉莲和赛莉耶能够在评论区隔一集（甚至隔一楼）抢到一次话筒，各自宣布一次喜欢花田魔法和讨厌花田魔法。一千个人能有一万个对花田魔法的锐评，但无论如何，虽然但是，花田对于原作 true end 的改编都是极为正确的。在 0 分的卷子上，无论改对了多少个答案，都比原来的分高。

Atri 动画最大的问题来自场外，出在原作党上。从来没有哪一部动画会有这么多的原作党出现，并像是列文虎克一样玩找不同游戏。例如，来自最后一集的一个 cut——男主与 Atri 二人一起在伊甸眺望亮起的小镇，当然，无疑这是在还原原作的 CG。原作党们经历了 12 集的洗礼，至此依然有时间精力发弹幕抨击“Atri 怎么没了”，却没有两三秒的时间等动画播放——三秒之后，Atri 就出现在画面中了。

像是这种的闹剧一样不少，在此引用泛式的比喻：Atri 的原作党们不是在看动画，而是在下棋，只是不停地讨论着每颗棋子应该放在哪。有些讽刺的是，Atri 动画播出前，大家还在恐怕“Atri 出圈了怎么办”这种问题。

不过，就算这样，Atri 依然是近年来表现最好的 gal 改动画之一了——其实近年来就没什么 gal 改，而 Atri 确实是今年唯一一部正片 gal 改动画。我们只能期待明年 Summer Pockets 动画的表现。


小市民シリーズ / 小市民系列

Bangumi 6.16 / Tascota 5.5

本作从头到尾似乎都像是对京阿尼的一次拙劣的模仿，尤其是我看本作的同时还同时在补《冰菓》动画，但京阿尼还做不出 21:9 画幅、推理全程不放 bgm 这类事情。意识流推理的意识仿佛关不上的水龙头，发了洪水，看起来莫名其妙。羊宫配的女主褒贬不一，但算是我看下去的唯一动力。

我会告诉入须学姐，这里有人很懂电影感，可以帮忙拍电影。


ダンジョンの中のひと / 地下城里的人们

Bangumi 6.52 / Tascota 6.5

本季度的百合豚避难所，地下城的 CGDCT 轻百合动画，有些贫穷，但评价方式依然是“认真你就输了”。

Belle 由铃代纱弓配音，稍稍可以代餐 Nijika。


時々ボソッとロシア語でデレる隣のアーリャさん / 不时轻声地以俄语遮羞的邻座艾莉同学

Bangumi 5.75 / Tascota Ø

《艾莉同学》的失利算是大女主轻小说的又一次意料之中的失败，当大家提到这部动画时，都只记得的是 Yuki 的“马达马达”，标题里用长定语修饰的“艾莉同学”就显得幽默了。

社友言：“比起改成动漫，不如改成 galgame，大女主人气不如妹妹”。


狠狠批斗组

しかのこのこのここしたんたん / 鹿乃子乃子乃子虎视眈眈

Bangumi 5.36 / Tascota 4.5

这其实是一部 SCP 基金会对于模因污染的警示片。如果你不懂什么是“模因”，那么看看《鹿乃子》里的“鹿”就知道了。

造成这种感觉的原因，在于《鹿乃子》根本上搞笑番失格。本作的搞笑差不多遵循这样的路径：有人开鹿、Torako 吐槽、笑点解析、旁白或者罐头笑声。全片笑点几乎无一例外都是这个公式本身或者它的变式，把开鹿或是换成妹妹发电，或是换成 Meme 吃饭。

多说无益，总之看完一整集《鹿乃子》的途中，能不走神的也都是神人了。


菜なれ花なれ / 亦叶亦花

Bangumi 5.23 / Tascota 4.0

PA 社本季三开之一。

尽管有独角兽老师的一定加持，本作的剧情依然不知所云，一头雾水，以至于让人不知从何下手吐槽。播到一半，弹幕大多还数不出主角团有几个人，更别提她们谁和谁是 cp、来自哪个学校；甚至就算到了最后一集，依然有我不认识的人渐次登场。

当然，本片中唯一行为较为正常的人类便是我罗，伟大无须多言。
