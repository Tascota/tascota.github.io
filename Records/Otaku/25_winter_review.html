Bangumi 标的完成数早就突破了一千，也认真地当 otaku 好几年了，其实早就感觉应该开始做些比较系统的总结了。虽然标注的大部分都是音乐，而我现在虽然学了些乐理的皮毛，但也还不会鉴赏音乐，于是就总结下追番好了。特别的部分是加上了对相关 anisong 的一些随意吐槽。总之因此，遂在今年夏季新番放送前，紧急赶制冬季漫谈。

只是到明年六月份为止都得备考，不知道能不能把这个习惯延续下去。

具体上说，大部分应该只是基于动画这个主题展开的漫谈而已，所以除了评分以外，评价部分可能不多。


Tier S

メダリスト / 金牌得主

Bangumi 8.00 / Tascota 8.0

除了马娘，我看过的唯一一部运动番就是《冰上的尤里》了。作为 BL 向，那是我的女性友人强行推荐的，其实老实说观感没有太好。但是《金牌得主》还是给了我很棒的观看体验。

具体关于运动本身的细节就不谈了，我也谈不了。为了观察这方面的细节，作为外行人，相当推荐看看 Bilibili 上曲教练的同步视听。自己看完后，有内行带着重看一遍，配上教练的相关杂谈补充，不仅能够理解细节，对花滑这项运动都能有更多的了解。

不过了解的过程也让我同样很难受。花滑训练中的“生长痛”——身体生长伴随着的疼痛，乃至“发育关”，也就是青春期的身体成长以及第二性征发育，对花滑选手的影响十分深刻。似乎不乏有选手因此终结了自己的比赛生涯。Inori 酱出场时是五年级，身高却只有 134 cm，怎么看都发育不良得过分。即使是动画后期时间有所跳跃，inori 也恐怕永远都不会像 momoka 一样有退出 U149 行列的风险了。但实际上，她在发育上的不足，却也可能是她能够取得如此成绩的一个优势——相比动画中着重描写的 ema 来说，显得更加明显。这其实不禁让我想到这项运动本身强调的方向是否有些偏移。

当然，以上都是我外行指导内行的发言，不过是站在道德高地上谴责罢了。

抛开这些沉重的话题不谈，动画本身对于角色的塑造——只和我上面看过的《尤里》对比，是相当充实的。运动番的重点是运动本身，而非功名化的比赛和成绩。

最后，inori 酱这么可爱，和 U149 里的 kaoru 一样（其实本质上是烤炉厨造就的一对异时空同位体），是在骗我们以后生女儿。这是一场贯通全东亚的巨大的阴谋。

    Tier A

    这个季度有太多我认为被严重低估的作品了。


    沖縄で好きになった子が方言すぎてツラすぎる / 在冲绳喜欢上的女孩方言讲得太过令人困扰

    Bangumi 7.00 / Tascota 7.5

    我认为这就是地方宣传番的标杆。相比起打卡一样的旅游，okitsura 把地方的风俗习惯揉碎，变成番剧的笑点或是情节的一部分，才能真正地让观众们能够留下印象。比如说，冲绳大街上的“比嘉”就像东京街上的“佐藤”一样多；贸然喊「比嘉さん」的话，因为谐音就是告白了。这种有趣的点才能让观众感受到一个地方的真正魅力。

    另外，通过教科书般可爱的两位女主，即使对冲绳风俗没有兴趣，这部番也能留住观众的视线。本番的绘柄极可爱，看到 kana 那双水灵灵的大眼睛，能不爱上的真是神人了。

    另外，我是黑皮控，这已经就是天堂。

    花は咲く、修羅の如く / 群花绽放，彷如修罗

    Bangumi 6.58 / Tascota 7.0

    武田大老师的又一部作品。

    其实看得出来，本番里有很多《吹》的影子，又是苦呀西，又是想要变得更好，又是轻百合（或者，台长说的轻 BG）。

    虽然即视感有些太严重了，但这部番依旧吸引我的原因在于，它确实非常“青春”。这一点武田老师还没输过。海边、小岛，JK 和全国大赛，哪有比这更青春的了？我在城里的高中，旁边的人倒是再怎么赛都没看出怎么青春。就算是 JK 斗财阀，都青春了起来。唯一缺憾在于其制作显然无法达到足够的表现力。

    另外，本作的绘柄也很好看。强烈推荐像小动物一样的 hana。

    空色ユーティリティ / 一杆青空

    Bangumi 6.78 / Tascota 7.0

    24 年一整年都没有什么萌豚动画摄入，我们萌豚差点就饿死了。一月一上来就有萌豚摄入，真是幸福。

    当然四月我们豚豚更是差点撑死了，这是后话。

    本番算是特别标准的 CGDCT，而且甚至是看得出来它是真的想让我们去打高尔夫的。但比起露营、爬山来说这个还要难多了。

    主角的 minami 酱是一只可爱的小狗狗，配色也很清爽，很可爱。

    Tier C


    もめんたりー・リリィ / 刹那之花

    Bangumi 4.41 / Tascota 4.0

    不知道在讲什么的动画。GoHands 的光污染画面更是打消了我任何想要认真看看这部番在讲什么的念头。虽然有人说后期回光返照了，但我的眼睛已经不同意我去好好看看它在讲啥了。

    主角团一行人，凭借翻来覆去的口头禅和单薄的塑造，无一不给人一种伪人感。这个季度是木偶们的主场。

    BanG Dream! Ave Mujica / 颂乐人偶

    Bangumi 4.12 / Tascota 3.5

    - 这是一部讲述人偶们奔向“没有剧本的世界”的动画。

    - 为什么要告诉我这个？

    - 没什么，只是想让你知道。

    关于 mujica 到底发生了什么，漫评 up 主讲得实在够多了。题外话，贝拉（也就是 asoul 的队长）的漫评倒是意外地切中肯綮，至少也是倪萍的一百万倍了。

    随便聊一下场外的话题。我们许多邦邦人只是因为这是邦邦的新动画所以要看，或者说因为 mygo 的动画确实不错所以继续看，多少都抱的是某种乐子人心态。但对于 mujica 的粉丝，乃至角色厨，实在难以想象这是一种多大的精神攻击。

    然后就是那惊为天人的剧透，我到现在都想不到是有一个多么草台班子的存在，才能在播不到一半的时候一直把到最后面的剧情都给透出来。反正应该不像是米哈游那样的炒作，众所周知，柿本以为自己写得太反转了，怎么会透出来。我当时看到时认为这种发展未免太违反动画创作的逻辑，一直认为这不可能是真实的剧情发展路线，结果自从 ep7 之后便一直跌破我的眼镜。自那之后，就从来没有人再认真地对待过这部动画的剧情。

    甚至最开始，在 ep1 开始，场外的大推理家们就应该认清自己身为折木奉太郎的现实的。停止作为武士道炒作的工具，认清真的就是剧本太菜的现实。不过这么说也是马后炮了，一声叹息。

    那场灾难性的“雨后天晴”之后，已经快过了四个月了。仔细想想，或许自从 mygo 动画里 kokoro 的消失开始，骰子就已经落下。
